# 纸牌屋字幕全文搜索站点

这是一个纯前端的《纸牌屋 (House of Cards)》字幕全文搜索小站点，支持在浏览器中对六季字幕进行英文台词全文检索，无需后端服务。

---


**[🚀 立即体验：https://stevenspage.github.io/scripts/ →](https://stevenspage.github.io/scripts/)**



---

## 功能特性

- **整句 / 单词搜索**：支持对整句英文台词或单词进行全文搜索。
- **通配符搜索**：支持使用 `*` 作为通配符，例如 `peach*`、`*bill`、`edu*bill`。
- **多季字幕**：目前已收录 **S01-S06 全部字幕**（`.lrc` 格式）。
- **纯前端实现**：所有字幕解析和搜索都在浏览器本地完成，可直接部署到 GitHub Pages 等静态托管服务。

---



---

## 搜索语法

- **普通搜索**：直接输入英文单词或短语，忽略大小写（如 `word travels fast in this town`）。
- **通配符 `*`**：
  - `peach*`：匹配以 `peach` 开头的单词。
  - `*bill`：匹配以 `bill` 结尾的单词。
  - `edu*bill`：匹配中间有任意字符的组合。
- **大小写敏感**：在搜索页右上角勾选“大小写敏感”可切换。
